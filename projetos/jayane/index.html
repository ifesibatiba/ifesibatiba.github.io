HTML

É uma linguagem de programação que aprendemos a utilizar no início do curso

Para que utilizamos o HTML?

O HTML é uma linguagem de programação de Hiper texto

que nos permite construir websites e a incersão de novos conteúdos,como imagens e vídeos

por meio de hiper texto

Projetos que aprendemos desenvolver:

Páginas, websites, sistemas

CSS

É uma linguagem de programação de disign gráfico que utilizamos dentro do código HTML

Para que utilizamos o CSS?

Ele nos permite aplicar estilos,como cores,margens, formas, tipos de letras e etc.

Css não é tecnicamente uma necessidade, mas provavelmente você não gostaria de olhar para um site

que usa apenas HTML, pois isso pareceria um site largado.

JAVASCRIPT

É mais uma linguagem de programação que aprendemos no final do curso.

Para que utilizamos o Javascript?:

Através dessa linguagem temos a possibilidade de colocar itens complexos em páginas web.

O que podemos fazer utilizando Javascript?

Desenvolver trabalho de montagem, realizar correções em sistemas para atender as necessidades dos

usuários, depuração e testes de programas já desenvolvidos
