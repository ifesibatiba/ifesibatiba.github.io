Scratch

“O Scratch é uma linguagem de programação criada pelo grupo Lifelong Kindergarten
da universidade americana MIT.
Tem como objetivo ensinar a lógica da programação para crianças e adolescentes.
Com ele, é possível criar histórias,
jogos e animações com scripts feitos com blocos.
É um tipo de programação “visual”, mais simplificada.
Isso permite que qualquer um, mesmo sem saber nada sobre programação,
consiga criar projetos digitais com mais facilidade.
