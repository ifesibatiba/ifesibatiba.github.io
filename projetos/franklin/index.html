HTML

É uma linguagem de programação que aprendemos a utilizar no início do curso

Para que utilizamos o HTML?

O HTML é uma linguagem de programação de Hiper texto que nos permite construir websites e a inserção de novos conteúdos, como imagens e víceos por meio de Hiper texto
