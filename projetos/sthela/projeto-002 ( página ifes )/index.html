O QUE APRENDEMOS?


HTML

(Linguagem de Marcação de Hiper Texto)

O HTML é a linguagem de marcação que nós usamos
para estruturar e dar significado para
o nosso conteúdo web.

Para que utilizamos o HTML?

O HTML é utilizado para desenvolver web.

Os desenvolvedores usam códigos HTML

para projetar como o navegador vai

exibir os elementos da página,

como textos, hiperlinks e arquivos de mídia.

Também para navegação na internet.

Os usuários podem navegar facilmente e inserir links

entre páginas e sites relacionados, já que o HTML

é amplamente usado para incorporar hiperlinks. Documentação.

O HTML torna possível a organização e a

formatação de documentos, de maneira similar ao Microsoft Word.

CSS

(Cascading Style Sheets)

O CSS é uma linguagem de regras de estilo que
nós usamos para aplicar estilo ao nosso conteúdo HTML.

Para que utilizamos o CSS?

O CSS pode ser usado para estilizar um documento
muito básico de texto — por exemplo,
alterando a cor e tamanho dos títulos e links.
Pode ser usado para criar layout — por exemplo,
transformando uma simples coluna de texto
em um layout com uma área de conteúdo principal
e um sidebar (barra lateral) para as informações relacionadas.
Pode até ser usado para efeitos tais como animação.

JS

(JavaScript)

O JS é uma linguagem de programação
que nos permite implementar itens
complexos em páginas web.

Para que utilizamos o JS?

JavaScript nos permite criar páginas da Web interativas e, portanto,
é uma parte essencial dos aplicativos da web.
A grande maioria dos sites usa, e todos os principais navegadores têm um mecanismo
JavaScript dedicado para executá-lo

Clusters, servidores de CDN, diminuir o tempo de requisição
