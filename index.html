Seja bem vindo ao nosso 1º projeto ( HTML/CSS )

Este projeto foi construido com a colaboração dos alunos do nosso curso de Programação para Internet.

O que se aprende em um curso de programação para internet?

Neste curso os alunos foram apresentados a diversas técnologias gratuitas, e boas práticas para programação, puderam entender o funcionamento da lógica através do Scratch, construiram uma pagina utilizando html puro, com css inline e por fim css externo.

Lógica com Scratch

Para conhecer o que é a lógica, e como ela se aplica a programação, utilizamos o Scratch, criaram script's, viram o que é uma repetição, criaram dinâmicas com o personagem.

Introdução ao HTML5 e CSS3

Foram feitas formatações e ajustes nos textos e imagens, apenas com html, porém como devemos seguir as boas práticas implantamos de forma externa o css, enriquecendo ainda mais nossos projetos.

Introdução ao JavaScript

Sabemos que hoje o JavaScript é uma das mais versateis linguagens de programação da atualidade, de fácil compreensão e com uma aceitação muito boa no mercado de trabalho, por isso não poderia ficar de fora.
